# 🍑 Butt Programming Language

Butt is a lightweight, meme-driven programming language built for fun, chaos, and educational suffering.

It runs on a simple Node.js interpreter and uses `.butt` files.

---

# 🚀 Features

- Custom `.butt` file extension
- Funny, human-readable syntax
- Simple CLI runner (`butt file.butt`)
- Lightweight JavaScript-based interpreter
- Built for learning + jokes + confusion 😄

---

# 📦 Installation

## 1. Clone the repository

```bash
git clone https://github.com/your-username/butt-lang.git
cd butt-lang
```

---

## 2. Install dependencies

```bash
npm install
```

---

## 3. Link CLI globally

```bash
npm link
```

---

# ▶️ Usage

Run a `.butt` file:

```bash
butt examples/main.butt
```

# 🧠 Syntax Guide

---

## 🔤 Variables

```butt
butt x = 10
```

Meaning:
```js
let x = 10
```

---

## 🖨️ Output

```butt
clap("Hello World")
```

Meaning:
```js
console.log("Hello World")
```

---

## 🔁 Conditions

```butt
vibe (x > 5) {
  clap("big energy")
} nah {
  clap("small energy")
}
```

Meaning:
```js
if (x > 5) {
  console.log("big energy")
} else {
  console.log("small energy")
}
```

---

## 🧠 Functions

```butt
cheek add(a, b) {
  drop a + b
}
```

Meaning:
```js
function add(a, b) {
  return a + b
}
```

---

## 🔁 Loops

```butt
wiggle (let i = 0; i < 3; i++) {
  clap(i)
}
```

Meaning:
```js
for (let i = 0; i < 3; i++) {
  console.log(i)
}
```

---

## 🔙 Return

```butt
drop x
```

Meaning:
```js
return x
```

# ⚙️ How It Works

Butt Lang:
1. Reads `.butt` files  
2. Replaces keywords with JavaScript  
3. Executes using Node.js
